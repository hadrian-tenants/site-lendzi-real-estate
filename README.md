# Lendzi Real Estate — Website

This site was generated by [Hadrian](https://hadrian.build).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see your site.

## Customization

Edit the components in `src/components/` to customize your site content.
Update `src/app/globals.css` to change brand colors.

## Deployment

This site is deployed automatically via Vercel. Push changes to the `main` branch and they will go live within seconds.

## Hadrian Integration

- **Apply page** — Embeds your Hadrian intake form so applicants can submit directly.
- **Status page** — Embeds your Hadrian status tracker so applicants can check progress.
- **API config** — `src/lib/hadrian.ts` connects to your Hadrian backend.
